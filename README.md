# COVID County SQL Analysis

## Overview
This project analyzes COVID-19 case and death trends across U.S. counties using SQL and Python to uncover geographic and temporal patterns.

## Objectives
- Identify counties with the highest case counts
- Compare state-level COVID impact
- Analyze national trends over time

## Dataset
- Source: Kaggle COVID-19 dataset
- Tables:
  - us_counties_daily
  - us_states_daily
  - us_national_daily

## Tools Used
- SQL (SQLite)
- Python (Pandas)
- Google Colab

## Key Questions
1. Which counties had the highest COVID-19 cases on the latest date?
2. Which states had the highest total cases and deaths?
3. What trends do we see over time in the U.S.?

## Key Insights
- Large urban counties (Los Angeles, NYC, Cook) had the highest case counts
- States like California, Texas, and Florida consistently ranked highest
- COVID cases and deaths showed strong upward trends over time

## How to Run
1. Open the notebook
2. Run all cells
3. Queries will execute automatically
